# MangoDongo
: 동국대학교 학생을 위한 음식점 추천 웹사이트

## 주요 기능
- 테마별 음식점 소개
- 음식점 소개 페이지
: 사용자 작성 리뷰, 평점, 위치, 대표 메뉴 등의 음식점 기본 정보
- 음식점 월드컵 페이지
- 회원가입, 로그인 후 리뷰 작성, 수정, 삭제


## 개발 환경
### 사용 IDE: 
- Eclipse
